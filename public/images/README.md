## Image Placeholders

The website includes image placeholders that will automatically display if images are missing. Once you add the images to this directory, they will automatically replace the placeholders.

## Image Guidelines

- Use optimized images (compressed for web)
- Recommended formats: PNG for logos, JPG for photos
- Maintain aspect ratios
- Logo should be transparent background PNG if possible
